# Screenshot Timer WinForms

Программа на C# Windows Forms для автоматического создания скриншотов каждые 5 секунд.

## Функциональность

- Запуск по кнопке "Пуск"
- Отображение прошедшего времени в заголовке окна
- Автоматическое создание скриншота каждые 5 секунд
- Сохранение скриншотов в формате BMP на диск D:
- Автоматическая остановка через 28 секунд (5 скриншотов)
- Использование Alt+PrintScreen для захвата активного окна

## Интерфейс

- Кнопка "Пуск" — запуск таймера
- Заголовок окна — отображение прошедших секунд
- Таймер с интервалом 1 секунда

## Пример работы

При запуске программа создаёт файлы:
- D:\Pic1.BMP (через 5 секунд)
- D:\Pic2.BMP (через 10 секунд)
- D:\Pic3.BMP (через 15 секунд)
- D:\Pic4.BMP (через 20 секунд)
- D:\Pic5.BMP (через 25 секунд)

Через 28 секунд программа автоматически закрывается.

## Куда можно встроить

- В программы мониторинга изменений на экране
- В демонстрационные и обучающие приложения
- В системы автоматической документации
- Как пример работы с таймером и буфером обмена

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Диск D: (или измените путь в коде)

## Примечание

Программа использует SendKeys для отправки комбинации Alt+PrintScreen и сохраняет изображение из буфера обмена. Для работы требуется активное окно, которое нужно захватывать.
